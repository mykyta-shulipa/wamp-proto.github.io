The Web Application Messaging Protocol

We're currently working on getting WAMP published as an IETF standard.

You can find the latest internet draft of the spec at the IETF website.

The source files for this are in the project GitHub repo.

If you want to participate in the development and standardization of WAMP, then see the contribution readme.
